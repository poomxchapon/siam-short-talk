Step 1 · วิเคราะห์บท

ต้องแยกให้ได้ 5 อย่าง

  1. ตัวละคร — ชื่อ อายุ ลุค บทบาท
  2. ฉาก — สถานที่ เวลา อากาศ
  3. Story — แต่ละ Story ≤ 30 วิ
  4. Shot — ตามจังหวะอารมณ์
  5. Lens — เลือกตามอารมณ์ shot

ตัวอย่าง — แบ่ง Shot

"สิงหา... แกแน่ใจแล้วเหรอ? 3 ปีที่ผ่านมา แกพิสูจน์อะไรได้บ้าง?"
A (3วิ)CU พ่อ 33mm → "สิงหา..."
B (2วิ)CU ลูก 85mm → reaction นิ่ง
C (3วิ)Wide 16mm → เห็น scale ห้อง
D (2วิ)Insert มือ 100mm → นิ้วเคาะ
E (2วิ)Low angle 24mm → ลูกไม่กลัว
F (3วิ)OTS 50mm → "แกพิสูจน์อะไร?"

Step 2 · Character Reference

สร้าง ref ตัวละคร front + side view สำหรับใช้ตลอด production

character reference sheet, front view and side view, Thai man, age 28,
handsome sharp jawline, short neat black hair, athletic build,
wearing worn-out plain black t-shirt,
neutral expression,
white background, ID photo style, clean lighting,
full face visible, photorealistic, Thai Asian face, consistent features
⚠️ ระวัง: Character ref sheet นี้ ห้ามใช้ตรงๆ กับ Seedance — มันจะ reject!
ใช้แค่กับ Nano Banana Pro เพื่อ gen ภาพฉากก่อน

Step 3 · Scene Reference

ภาพฉาก ไม่มีคน เพื่อใช้เป็น ref ให้ Seedance

Cinematic wide shot, 16mm lens,
grand white neoclassical European mansion at dusk,
tall white columns, arched windows, white marble facade,
warm golden lights glowing from windows,
circular driveway with elegant fountain,
dark dramatic clouds in sky,
no people,
photorealistic, vertical 9:16

✅ ต้องมี

  • เลนส์ชัดเจน (16mm / 33mm / 50mm)
  • ช่วงเวลา sync กับฉากอื่น
  • no people ทุกครั้ง
  • vertical 9:16

💡 Trick

Interior หลายมุม → ใช้ multiple angles, 4 panels layout จะได้ 4 มุมในรูปเดียว เป็น ref ได้หลาย scene

Step 4 · Video Gen — Seedance 2.0

กฎเหล็ก: Seedance ใช้ ภาษาจีน เสมอ — ได้ผลดีกว่า EN หลายเท่า 
@[ref สิงหา] นี่คือ สิงหา ,
@[ref ฉาก] นี่คือ ref ของฉากนี้
cinematic live-action video，Netflix 原创剧集质量，
85mm镜头特写，สิงหา穿褪色黑色T恤，
自信温暖的微笑，眼神坚定充满爱意，
idle状态自然呼吸轻微眨眼，
豪宅客厅水晶吊灯金色光斑虚化背景，
温暖金色灯光，极浅景深，竖屏9:16

Tip: ตั้งชื่อตัวละครไทยตรงๆ นี่คือ สิงหา ดีกว่าเขียน "young Thai man age 28"

Idle vs Talking

Rule of thumb: ทุก shot ที่มีบทพูด → gen เป็น idle ก่อน แล้วค่อยทำ lipsync แยก
เพราะ Seedance ขยับปากพูดไม่แม่น

😐 Idle (ไม่ขยับปาก)

idle状态，
自然呼吸，
轻微眨眼，
极轻微晃动，
保持静止

🗣️ Talking (ขยับปาก)

嘴唇自然开合说话，
头部轻微点头强调重点
正在说话：
"[บทพูดภาษาไทย]"

เลนส์ = ภาษาอารมณ์

เลือกเลนส์ตาม อารมณ์ที่ต้องการสื่อ ไม่ใช่แค่ระยะใกล้/ไกล

Lensฟีลใช้ตอน
16mmกว้าง เห็น scale อำนาจEstablishing, ห้องใหญ่ คนตัวเล็ก
33mmCU + เห็น contextเปิดฉาก, บทสนทนาที่ต้องเห็นห้อง
50mmตาคนจริง ธรรมชาติOTS, POV, tracking
85mm ⭐ ใช้บ่อยBokeh หนัก isolate อารมณ์Reaction, emotional CU
100mm macroDetail จัดสุดInsert — มือ, วัตถุ, detail
135mmเบลอหมด เหลือจุดเดียวECU ตา, น้ำตา, tension สูงสุด

ตัวอย่าง — เลือก Lens ตามอารมณ์

ฉากพ่อท้าลูก (กดดัน)

CU พ่อ 33mm (เห็น context ห้อง)
→ CU ลูก 85mm (isolate อารมณ์)
→ Wide 16mm (เห็น scale ห้องใหญ่ ลูกตัวเล็ก)
→ Insert มือ 100mm (นิ้วเคาะเก้าอี้)
→ Low angle ลูก 24mm (ลูกไม่กลัว)
→ OTS 50mm (มุมตาลูกมองพ่อ)

ฉากดราม่าฝนตก (เจ็บปวด)

Full body 35mm (เห็นทั้งคน+ฝน+บ้าน)
→ CU หน้า 85mm (น้ำตา+ฝน)
→ Insert เค้กในโคลน 100mm (detail ความเจ็บ)
→ ECU ตา 135mm (เบลอหมด เหลือแค่ตา)

หลบ Content Filter ⚠️ สำคัญ

Seedance มี moderation เข้ม — ต้องรู้วิธีหลบ

🖼️ Image Filter

Character ref มีอาวุธ/เกราะ/ท่าก้าวร้าว → reject

  • Error: 你上传的图片不符合平台规则
  • Fix: Crop เฉพาะหน้า หรือใช้ภาพฉากแทน
  • Transition clip → แนบแค่ scene ref

📝 Text Filter

คำต้องห้าม → คำแทน

❌✅
凶狠 (ดุร้าย)严肃 / 坚定
威胁 (ข่มขู่)不满
血/死/杀痕迹 / 裂纹
爆炸能量波纹
ตาย/ฆ่าเก็บไว้ TTS

Step 5 · Voice Design — ElevenLabs

Voice Design prompt = ออกแบบเสียงด้วยภาษาเขียน

ตัวอย่าง — พ่อผู้มีอำนาจ

An elderly Thai man in his mid-60s.
Deep, authoritative baritone voice with a commanding presence.
Speaks slowly and deliberately, each word carrying weight and power.
Gravelly texture from age but still strong.
Calm but intimidating. Slight rasp.
Long pauses between sentences.
Structure: เพศ + อายุ → ลักษณะเสียง → วิธีพูด → อารมณ์ → ลักษณะพิเศษ

Common Mistakes

❌ ผิด✅ ถูก
ใช้ face ref sheet ตรงๆ ใน SeedanceGen ภาพฉากก่อน → ใช้ภาพฉากเป็น ref
Gen ปากขยับพูดGen idle → lipsync แยก
Intercut 5 มุมใน 5 วินาที1 clip = 1 มุม 1 action → ตัดต่อรวม
เขียน prompt EN ใน Seedanceใช้จีน — ผลดีกว่าหลายเท่า
ใช้เลนส์เดิมทุก shotสลับเลนส์ตามอารมณ์
ช่วงเวลาไม่ sync (กลางคืน→เช้า)เช็ค continuity ทุก cut
ไม่ใส่ no background musicใส่ไว้ ไม่งั้นมันเติมเพลงเอง

กฎทองที่ต้องจำ

ถ้า Seedance ต้องเลือกระหว่าง
text prompt กับ ref image
มันจะเชื่อ text มากกว่า
แปลว่า: ถ้าภาพ ref โทนแดง แต่ prompt เขียน cool blue → ได้สีขัดกัน
How to apply: ตรวจ prompt ให้ sync กับ ref image ทุกครั้ง

Workflow จริง · End-to-End

1 อ่านบท → แบ่ง shot + lens + timeline
2 Gen character ref (Nano Banana) → download
3 Gen scene ref (Nano Banana) → download
4 Seedance — แนบ ref → gen idle 5 วิ/clip → download
5 ถ้าติด filter → ปรับ prompt → retry
6 ElevenLabs → voice design → TTS บทพูด
7 Lipsync tool → video + audio
8 Premiere/CapCut → ตัดต่อ + SFX + เพลง + ซับไทย
9 Export 1080×1920 (9:16) → IG Story / TikTok / Reels
